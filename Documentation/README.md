## Contributing

Yapper follows a modular design. When adding new features:
1.  Define the default configuration in `Core.lua`.
2.  Add the corresponding UI controls in `Interface.lua`.
3.  Implement the logic in a new `Src/` module and register it on the `YapperTable`.
4.  Update this documentation to reflect the changes.
